# 🍽️ Recipe Page

Projeto de uma página de receita desenvolvido com HTML e CSS como parte do meu aprendizado em desenvolvimento web.

## 📅 Dia 1 - Estrutura HTML

Neste primeiro dia, foi criada a estrutura base da página utilizando HTML5, aplicando:

* Estrutura semântica (`<main>`, `<section>`, `<footer>`)
* Organização do conteúdo da receita
* Uso de listas (`<ul>`) para ingredientes
* Separação de seções: sobre, ingredientes e modo de preparo

## 🎨 Dia 2 - Estilização com CSS

No segundo dia, evoluímos a interface adicionando estilos e refinando o layout:

* Aplicação de fonte personalizada ("Alice")
* Adição de imagens (imagem principal e background)
* Definição de cores e identidade visual
* Criação de um container central estilizado (`#page`)
* Uso de `padding` e `margin` para organização do layout
* Ajustes de alinhamento entre imagem e conteúdo (hierarquia com `<main>`)
* Bordas arredondadas em imagens e elementos
* Ajustes de tipografia (títulos e textos)
* Organização das informações da receita (tempo, rendimento e dificuldade)
* Correção de espaçamentos e consistência visual

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## 📌 Objetivo

Praticar a construção de páginas estruturadas e evoluir na estilização com CSS, aplicando conceitos de layout, espaçamento e hierarquia de elementos.
