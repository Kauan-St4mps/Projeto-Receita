🍽️ Recipe Page
Projeto de uma página de receita desenvolvido com HTML como parte do meu aprendizado em desenvolvimento web.

📅 Dia 1 - Estrutura HTML
Neste primeiro dia, foi criada a estrutura base da página utilizando HTML5, aplicando:

Estrutura semântica (<main>, <section>, <footer>)
Organização do conteúdo da receita
Uso de listas (<ul>) para ingredientes
Separação de seções: sobre, ingredientes e modo de preparo

🛠️ Tecnologias utilizadas
HTML5

📌 Objetivo
Praticar a construção de páginas estruturadas e semânticas, criando uma base sólida para estilização com CSS nos próximos passos.
